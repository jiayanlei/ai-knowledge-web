# 企业 AI 知识库助手前端

企业内部 AI 知识库与员工助手平台前端骨架，基于 Vue 3、TypeScript、Vite、Element Plus、Pinia、Vue Router、Axios、ECharts、Sass 和 UnoCSS。

## 本阶段范围

- 静态路由与基础布局
- AI 问答三栏页面
- 页面内部 mock 数据展示
- 不包含真实登录、真实接口、真实权限和真实上传

## 启动

```bash
npm install
npm run dev
```

默认访问：

```text
http://localhost:5173
```

`/` 会重定向到 `/ai-chat`。
